# Brand Core

Ядро бренда Rashamon — ценности, позиционирование, идентичность.

---

## Name Origin

**Rashamon** (羅生門) — отсылка к фильму Акиры Куросавы «Расёмон» (1950).

Фильм исследует **субъективность истины** — одно событие через глаза разных участников выглядит по-разному.

### Почему это имя

| Связь | Rashamon (фильм) | Rashamon (продукт) |
|---|---|---|
| Множество перспектив | 4 версии одного события | Branching history — множество версий проекта |
| Субъективность | Каждый видит своё | Каждый workflow уникален |
| Глубина | Не surface-level история | Не surface-level инструмент |
| Японское происхождение | Куросава, Акутагава | Отсылка к японской культуре качества |

## Brand Architecture

```
Rashamon (umbrella brand)
├── Rashamon Draw
├── Rashamon Photo
└── Rashamon Motion
```

Каждый продукт:
- Использует имя Rashamon + suffix
- Имеет уникальную иконку/цвет
- Следует общим brand guidelines

## What Rashamon Brand Is NOT

| Не является | Почему |
