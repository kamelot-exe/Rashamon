# Why Rashamon Exists

> Креативные профессионалы на Linux заслуживают инструментов, которые сделаны для них — не портированы задним числом, а спроектированы с нуля.

---

## The Gap

### Linux — третья платформа по вниманию, первая по потенциалу

- **~3% десктопного рынка** — но это миллионы разработчиков, дизайнеров, системных администраторов
- **0% внимания** от крупных вендоров креативного софта
- Adobe, Figma, Affinity — ни одного нативного Linux-клиента
- Существующие open-source инструменты (Inkscape, Krita, GIMP) — отличные проекты, но с **архитектурным долгом** 10+ лет

### Open-source креативный софт застрял в парадигме «клон»

- Inkscape → клон Illustrator
- Krita → клон Photoshop
- GIMP → клон Photoshop
- Blender → исключение: нашёл **свою** нишу (3D) и доминировал в ней

**Нет open-source инструмента, который бы предложил уникальный workflow**, а не повторил чужой UI.

---

## Почему сейчас

1. **Tauri 2** — зрелый фреймворк для desktop-приложений на web-технологиях с Rust-бэкендом
2. **React + TypeScript** — де-факто стандарт для сложных UI
3. **SQLite** — встраиваемая БД для структурированного хранения проектов
4. **Рост Linux-десктопа** — Steam Deck, системные дистрибутивы,开发者 adoption
5. **Усталость от подписки** — рынок ищет альтернативы perpetual-license и SaaS

---

## Что даёт Rashamon

### Для пользователей
- Профессиональный инструмент **без подписки**
- **Нативная** работа на Linux
- **Открытый** код — можно проверить, форкнуть, исправить
- **Уникальные фичи**: семантический канвас, графовая история, автоматизация

### Для контрибьюторов
- Чёткая архитектура и документация
- Модульная структура — можно контрибьютить в отдельный модуль
- Governed open source — баланс между открытостью и целостностью бренда
- AI-ассистированная разработка (multi-agent workflow)

### Для экосистемы
- Общая платформа для нескольких продуктов
- Плагин-система с реальным API
- Формат проектов, который можно использовать независимо

---

## Почему не «ещё один клон»

Клон — это **стратегия догоняния**. Доверяющий клон:

- Всегда на шаг позади оригинала
- Не предлагает причины мигрировать, кроме «бесплатно»
- Копирует чужие ошибки UX вместе с фичами
- Не создаёт комьюнити — создаёт базу пользователей

Rashamon — это **стратегия дифференциации**:

- Семантический канвас вместо плоского холста
- Графовая история вместо линейного undo
- Неразрушающее редактирование как философия, а не опция
- Автоматизация и плагины как первоклассные граждане
- Linux-native мышление, а не port

---

## Key Decisions

| Решение | Почему |
|---|---|
| Экосистема, а не один продукт | Общая платформа amortizes成本; каждый продукт усиливает остальные |
| Начать с Draw | Векторный редактор — наименьший MVP, максимальная архитектура |
| Не копировать Adobe UI | Наша цель — улучшить workflow, а не повторить чужой |
| Бренд-управление отдельно от лицензии | Код открыт, имя защищено — как у Mozilla, Blender, Red Hat |
