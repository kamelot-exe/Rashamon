# Contributing to Rashamon

Thank you for your interest in contributing to Rashamon!

We welcome contributors who care about architecture clarity, performance, maintainability, and serious open-source collaboration.

1. **Read the vision** — [Project Vision](./docs/vault/01_VISION/Project%20Vision.md)
2. **Check the roadmap** — [Product Roadmap](./docs/vault/02_STRATEGY/Product%20Roadmap.md)

4. **Read the guide** — [Full Contribution Model](./docs/vault/07_COMMUNITY/Contribution%20Model.md)
